# S.E.P
Music App

-----
> **USEAGE**

### Build:

  ```
  $ source .env
  ```

  - Using <a href="https://www.docker.com/">Docker</a>  

    ```
    $ docker-compose up
    ```
  - Open Terminal
    ```
    $ docker exec -it mongodb bash (To open execution path of mongodb in docker image )
    $ mongo -u admin -p admin --authenticationDatabase musicapp 
    ```

  - Run Server:
    ```
    $ localhost:5000
    ```
